== Description ==

Le badge Google+ par JM Créa permet d’afficher le badge de votre compte Googtle+ n’importe où (pages et/ou posts). 

ce plugin est paramétrable, vous pouvez choisir l’affichage vertical, horizontal, sombre, clair, choisir la taille et bien sûr, choisir l’url de votre compte.

Pour appeler ce plugin sur vos pages, utilisez le shortcode [bgp_by_jm_crea]

== Installation ==

1. Uploadez `googleplus-badge-by-jm-crea` dans le dossier `/wp-content/plugins/` de votre site
1. Activez ’Google+ Badge par JM Créa’ depuis le menu « Extensions » de Wordpress
1. Rendez-vous ensuite dans l’onglet ‘Outils -> Google+ Badge’

1. Vous pouvez aussi appeler votre badge avec le shortcode : `[bgp_by_jm_crea]`
